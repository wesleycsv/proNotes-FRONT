## Iniciar o FRONT

Para inicar a api e preciso primeiro instalar os pacotes

```sh
yarn ou npm install
```

Depois baixe a API 
https://github.com/wesleycsv/proNotes-API
Siga as instruções e inicie a API para depois iniciar o FRONT

Agora so é iniciar o projeto

```sh
yarn dev
```
